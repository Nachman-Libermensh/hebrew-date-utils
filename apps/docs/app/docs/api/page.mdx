# hebrew-date-utils (Core API)

`hebrew-date-utils` is the primary library in this repository.

It is designed for date-only Hebrew/Gregorian workflows in any Node.js project:

- Backend services
- Workers and schedulers
- CLI tools
- Serverless/API routes

Under the hood it uses:

- `@hebcal/core` for Hebrew calendar rules and conversions
- `date-fns` for Gregorian arithmetic and formatting

## Common Imports

```ts
import {
  toDualDate,
  toGregorian,
  toHDate,
  listDaysInHebrewMonth,
  getGregorianMonthBoundaries,
  differenceInDualDays,
  differenceInDualMonths,
  addGregorianYears,
  getHolidaysOn,
} from "hebrew-date-utils";
```

## Module Highlights

### Conversions

Use `toDualDate`, `toHDate`, and `toGregorian` to normalize mixed inputs.

### Boundaries and Ranges

Use `getGregorianMonthBoundaries`, `getHebrewMonthBoundaries`, and `listDualDatesInRange`.

### Arithmetic

Use `differenceInDualDays`, `differenceInDualMonths`, `differenceInDualYears`, and Hebrew-specific month/year differences.

### Navigation

Use `addGregorianDays`, `addGregorianMonths`, `addHebrewMonths`, and related helpers to move in either calendar.

### Holidays

Use `getHolidaysOn`, `getHolidaysBetween`, and `getHolidaysForHebrewYear`.

## End-to-end example

```ts
import {
  toDualDate,
  listDualDatesInRange,
  getHolidaysBetween,
  splitRangeByHebrewMonth,
} from "hebrew-date-utils";

const start = new Date(2026, 8, 1);
const end = new Date(2026, 9, 15);

const days = listDualDatesInRange(start, end);
const segments = splitRangeByHebrewMonth(start, end);
const holidays = getHolidaysBetween(start, end, { locale: "he", il: true });

console.log({
  first: toDualDate(days[0]!.greg),
  totalDays: days.length,
  monthSegments: segments,
  holidays,
});
```
